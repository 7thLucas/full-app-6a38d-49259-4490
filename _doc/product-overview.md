# WanderMind — Product Overview

## What it is
WanderMind is an AI-powered travel planner that acts as a personal travel agent. Users describe their destination, budget, and interests in natural language; the AI generates a complete, personalized itinerary — tourist spot recommendations, hotel suggestions, daily activity schedules, and an adaptive plan that updates based on user feedback.

## Core problem it solves
Planning a trip is time-consuming and fragmented. Travelers spend hours across multiple tabs — researching destinations, comparing hotels, mapping routes, and assembling a day-by-day schedule manually. There is no single tool that does all of this intelligently and adapts to what the user actually wants.

## Solution
WanderMind replaces manual research with a conversational AI agent. A user inputs destination, budget, duration, and personal interests; the AI returns a full itinerary instantly. The plan is displayed in two ways: a **timeline view** (day-by-day schedule) and an **interactive map** (all locations pinned and routed). Users can give feedback ("more budget options", "fewer museums", "add nightlife") and the AI adjusts the plan in real time.

## Target users
- Solo travelers wanting fast, personalized trip planning
- Couples and families planning vacations without a travel agency
- Backpackers with tight budgets who need cost-optimized itineraries
- Business travelers adding leisure time to work trips

## Key features
1. **AI Chat Interface** — conversational input for destination, budget, duration, interests
2. **Full Itinerary Generator** — complete day-by-day plan with morning/afternoon/evening slots
3. **Hotel Recommendations** — curated options filtered by budget and location
4. **Tourist Spot Discovery** — AI-ranked attractions matched to user interests
5. **Daily Activity Planning** — time-blocked schedule with logistics (distance, duration)
6. **Timeline View** — visual chronological layout of the entire trip
7. **Interactive Map** — all locations pinned; routes visualized
8. **Feedback Loop** — user can refine the plan via follow-up chat; AI adjusts instantly

## Brand identity
- **Name**: WanderMind
- **Tagline**: Plan Less. Explore More.
- **Visual identity**: Teal and ocean blue gradient; AI compass/globe motif; clean and modern
- **Tone**: Smart, adventurous, efficient — feels like a brilliant friend who knows travel

## Positioning
Personal AI travel agent for independent travelers. Faster than a travel agency, smarter than a search engine, more personal than a generic itinerary app.

## Scope (MVP)
- AI chat for trip input and feedback
- Itinerary generation with hotel + attraction + activity recommendations
- Timeline view
- Interactive map
- Plan refinement via conversation

## Strategic principles
- Every plan adapts to the user — no generic outputs
- Display both timeline and map simultaneously; one without the other is incomplete
- Budget awareness is core — recommendations must respect the stated budget
- The feedback loop is a first-class feature, not an afterthought
